# Project---Customer-Analytics-Dashboard-tableau
Designed and developed a multi-view interactive dashboard in Tableau Public to analyze customer purchasing behavior, regional revenue distribution, and payment preferences across the United States. The dashboard consolidates complex transactional data into clear, actionable visual insights for business stakeholders.
